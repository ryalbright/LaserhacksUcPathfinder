# LaserhacksUcPathfinder
UC Pathfinder is a website that helps new students find the best UC to transfer into from IVC for their major, using transfer rates. The pathfinder also finds the average salary of graduates of that major from the university, and provides clubs that may help the user in pursuing their selected major at IVC. Requires Python 3.10 and Flask.
