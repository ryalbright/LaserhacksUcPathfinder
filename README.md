# LaserhacksUcPathfinder
UC Pathfinder is a website that helps new students find the best UC to transfer into from IVC for their major, using transfer rates. The pathfinder also finds the average salary of graduates of that major from the university, and provides clubs that may help the user in pursuing their selected major at IVC. The pathfinder also gives people target majors based on their projected GPA that give the highest return on investment. Requires Python 3.10 and Flask.

Ryan Albright - Handled Backend, Salary Data, UC Statistics, dropdown menus, data_processing.py, and GPA ROI major finder.
Ryan Duncan - Handled Club Matching, ClubData.csv, and Frontend

To run, run backend.py
